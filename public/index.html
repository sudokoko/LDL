                        Block people you don't like.

                        LBP Union Engineer Zaprit coming to the rescue, completing work on a much-needed
                        Lighthouse Feature. You will now be able to block users that are causing you trouble,
                        which will prevent them from doing a few things.

                         * Reviewing your levels
                         * Commenting on your Profile
                         * Joining you in-game (potentially)

                        This will also prevent you from being able to view that user's comments.

Project Lighthouse is a clean-room, open-source custom server for LittleBigPlanet.

asp-net lighthouse lbp private-server littlebigplanet custom-server
Project Lighthouse is not production-ready. We currently do not provide support for the project. All source code is provided as-is, without warranty. You have been warned.
